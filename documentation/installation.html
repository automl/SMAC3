Installation¶

Requirements¶

SMAC3 requires:

  • NumPy (Version >= 1.6.1)
  • SciPy (Version >= 0.13.1)
  • pynisher (Version >= 0.4.1)
  • ConfigSpace (Version >= 0.2.0)
  • setuptools
  • six
  • Cython
  • pyrfr

Manual Installation¶

To install EPM from command line type the following commands in the terminal:
cat requirements.txt | xargs -n 1 -L 1 pip install
python setup.py install
