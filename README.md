# SMAC3



## Installation
```
git clone https://github.com/https://github.com/automl/SMAC3.git
cd SMAC3
conda create -n smac python=3.8
conda activate smac

# Install for usage
pip install .

# Install for development
make install-dev
```

Documentaiton at https://https://github.com/automl.github.io/SMAC3/main
